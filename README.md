## About Me

## Features
- **Welcome Section:** A simple and attractive welcome message with a "Learn About Me" button.
- **About Me Section:** Includes profile image, brief introduction, hobbies, and contact information.
- **Dropdowns:** Collapsible sections for hobbies and contact details.
- **Back Button:** Allows returning to the welcome section smoothly.
- **Interactive Animations:** Smooth fade-in/out transitions, button hover effects, and image zoom.

## Technologies Used
- **HTML5** - Structure of the webpage
- **CSS3** - Styling and animations
- **JavaScript** - Interactive elements (dropdowns, navigation)
- **Responsive Design** - Works well on different screen sizes
